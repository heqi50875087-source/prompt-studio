# 通用电影级英文分镜 · 知识包

> 用途：把任意一个中文创意，按固定字段翻译成跨平台（即梦 / Runway / Kling / Veo / Sora / Pika 通用）的电影级英文提示词。
> 适用人群：少儿内容创作（童话 / 寓言 / 绘本短片）为主，方法对所有题材通用。
> 怎么用：① 看「五条铁律」建立直觉 → ② 按「Schema」给每个镜头填 10 个字段 → ③ 从「英文词表」挑 token 拼成英文 → ④ 参照「三组示例」照葫芦画瓢 → ⑤ 需要溯源时查「资源库」。

## 五条铁律（所有平台通用，最高价值）

1. **镜头与主体分开写**：先写镜头（景别+机位+运镜+镜头规格），再写主体，二者不要混在一句里——这是跨平台最稳的写法。
2. **固定顺序铺陈**：`Camera → Subject → Action → Setting → Lighting → Style → Format/Negative`，写 2–4 句最稳。
3. **动词要具体**：写 `opens the umbrella` / `walks to the door and knocks`，不要写 `moves`、`camera moves through scene`（最常见翻车点）。一个镜头只放**一个主体**。
4. **至少两种色温**：每条电影感 prompt 给暖 vs 冷（或硬 vs 软）的光对比——这正是「电影感」的来源。
5. **风格做成固定 style block**：把 `cinematic, color grade, film/镜头, depth of field…` 固化成一段，贴到全片每个镜头末尾，保证整组风格统一。先用文生图定一张静帧做「锚点」，再图生视频，跨镜一致性最好。

> 少儿专属补充：角色要友好（大眼、不露利齿），风格优先「3D 绘本动画 / 手绘水彩」，负面词务必加 `scary, violent, dark horror, sharp teeth`。

## 一、通用分镜 Schema（每个镜头 10 字段）

| 字段 | 英文 | 写什么（要点） | 例 |
|---|---|---|---|
| 镜号 | shot no. | 顺序编号 | S1 / S2 |
| 景别 | **shot** | 主体在画面里的大小/距离 | wide shot / close-up |
| 机位+运镜 | **camera** | 角度 angle + 运动 movement + 速度 +（镜头规格 lens） | low angle, slow dolly-in, 35mm |
| 主体 | **subject** | 一个主体 + 辨识特征（颜色/服装/表情），只放一个 | a little girl in a red hooded cape |
| 动作 | **action** | 一句具体动词短语，写清从头到尾 | walks along the path and looks around |
| 场景 | **setting** | 地点 + 时间 + 天气 + 道具 | sun-dappled forest, morning |
| 光线 | **lighting** | 光质 + 方向 + 色温（至少冷暖各一） | warm golden light, cool sky rim |
| 风格 | **style** | 类型 + 调色 + 质感 + 胶片/镜头（做成固定 style block） | soft 3D storybook, shallow DoF |
| 时长 | **duration** | 秒数（受模型上限：Veo 4/6/8；Kling 5–10；Sora 4/8/12/16/20） | 5s / 8s |
| 画幅 | **aspect** | 比例 | 16:9 / 9:16 / 1:1 / 2.39:1 |

**拼装成一条英文 prompt 的万能公式：**

```
[景别 shot + 机位 angle + 运镜 movement + 镜头 lens] of [主体 subject + 辨识特征],
[动作 action: 具体动词短语]. [场景 setting: 地点+时间+天气+道具].
[光线 lighting: 光质+方向+冷暖]. [风格 style block: 类型+调色+质感].
[画幅 aspect], [时长 duration]. Negative: [不想要的东西].
```

**JSON 结构化变体**（Veo 支持，能防「concept bleed」描述情绪误改物体颜色）：

```json
{ "shot":"medium close-up", "subject":"...", "action":"...",
  "camera":"slow dolly in", "lighting":"warm golden hour, rim light",
  "style":"cinematic, film grain, teal-orange grade",
  "aspect_ratio":"16:9", "duration":"8s", "negative":"blurry, text, watermark" }
```

**两套官方分段框架（与上面同构，可任选）：**

- **Veo 公式**：`[Cinematography] + [Subject] + [Action] + [Context] + [Style & Audio]`
- **V.I.D.E.O. 五段填空**：`V` Visuals(场景+时段+镜头风格) · `I` Intent(目标+主焦点) · `D` Details(光+环境+运动+道具) · `E` Emotion(主+次情绪) · `O` Output(时长+画幅+画质)
- **Sora 2 官方五段**：`Scene description`(人物/服装/布景/天气) · `Cinematography`(景别+机位+运动+镜头规格，单独成段) · `Actions`(编号 beat：1)…2)…) · `Dialogue`(简短台词) · `Ambient sound`。注意 Sora 的**时长与画幅是 API 参数**（`seconds` 与 `size`），不写进正文；适度留白反而鼓励模型发挥。

> 提示：即梦 / Kling 的「镜头语言」多在**菜单栏勾选**，不必写进文本框；其余平台才把镜头写进正文。

## 二、镜头语言英文词表（挑 token 拼即可）

### 1. 景别 Shot type（由近到远）

| 英文 token | 中文 | 用途/效果 |
|---|---|---|
| extreme close-up (ECU) | 极特写 | 细节/张力 |
| macro close-up | 微距特写 | 极小物体细节（露珠、萤火） |
| close-up (CU) | 特写 | 表情/细微情绪 |
| medium close-up (MCU) | 近景（脸+肩） | 对话情绪 |
| medium shot (MS) | 中景（腰以上） | 手势+表情，中性叙事 |
| full shot / full body | 全身 | 完整动作 |
| wide shot (WS) | 远景 | 主体+环境 |
| establishing (wide) shot | 建场镜 | 开场交代地点 |
| extreme wide shot (EWS) / aerial | 大远景/航拍 | 史诗感、孤独尺度 |
| two-shot | 双人镜 | 表现两者关系 |
| over-the-shoulder (OTS) | 过肩 | 对话临场感 |
| point-of-view (POV) | 主观视角 | 强代入 |
| insert / cutaway | 插入/空镜 | 补细节 |

### 2. 机位/角度 Camera angle

| 英文 token | 中文 | 效果 |
|---|---|---|
| eye-level | 平视 | 中立、自然、平等 |
| low angle | 仰拍 | 强大/英雄/压迫 |
| high angle | 俯拍 | 弱小/脆弱 |
| bird's-eye / top-down / overhead | 上帝视角/俯瞰 | 地理尺度、布局 |
| worm's-eye | 虫视/极低 | 夸张高耸 |
| dutch / canted angle | 倾斜机位 | 不安/失衡/混乱 |
| three-quarter angle | 四分之三侧 | 立体、自然 |
| over-the-shoulder (OTS) | 过肩 | 对话 |

### 3. 运镜 Camera movement（最易翻车，给齐 type+speed+direction+起止）

| 英文 token | 中文 | 效果 |
|---|---|---|
| static / locked-off | 固定 | 稳定、克制 |
| pan left/right | 横摇 | 水平扫视 |
| tilt up/down | 纵摇 | 垂直扫视 |
| dolly in / push-in | 推 | 建张力（物理推近，景深自然） |
| dolly out / pull-back (reveal) | 拉/揭示 | 拉开揭示环境 |
| truck left/right | 横移 | 平行移动 |
| pedestal up/down | 机身升降 | 垂直升降 |
| tracking / following shot | 跟拍 | 跟随主体 |
| arc / orbit | 环绕 | 绕主体（hero reveal） |
| crane / jib / boom | 摇臂 | 大幅升降 |
| zoom / crash zoom | 变焦/急推 | 改焦距（≠景深，区别于 dolly） |
| whip pan | 甩镜 | 快速转场 |
| dolly zoom / vertigo | 滑动变焦 | 眩晕、心理失衡 |
| rack focus | 变焦点 | 焦点在前后景间转移 |
| steadicam | 稳定器 | 平滑跟随 |
| handheld | 手持 | 轻微抖动，真实/紧张 |
| drone / aerial flyover / FPV | 无人机/航拍 | 大场面 |
| **速度词** | — | slow / gentle / steady / moderate / fast |

> 进阶两招：① 把 CAMERA 指令单独成段（全大写标签）并写明起止位姿；② 给「动机」motivated movement，如 `slow dolly in as she reads the letter — intimacy building`。
> 示例：`CAMERA: HALF ORBIT. Camera travels in a curved arc 180° around the subject on a consistent radius. Begins at a frontal view, ends at a rear three-quarter angle. Slow, steady speed.`

### 4. 光线 Lighting

| 英文 token | 中文 | 效果 |
|---|---|---|
| golden hour / magic hour | 黄金时刻 | 暖、柔、电影感 |
| blue hour | 蓝调时刻 | 冷、宁静 |
| three-point lighting (key/fill/back) | 三点布光 | 标准人像 |
| Rembrandt lighting | 伦勃朗光（颊上三角） | 戏剧人像 |
| chiaroscuro | 明暗强对比 | 黑色电影/张力 |
| split lighting | 分割光 | 一半亮一半暗 |
| rim / back light, backlit | 轮廓光/逆光 | 勾边、与背景分离 |
| silhouette | 剪影 | 神秘 |
| soft diffused light | 柔光 | 温柔/梦幻（少儿首选） |
| hard light | 硬光 | 强反差 |
| high-key | 高调 | 明亮/欢快 |
| low-key | 低调 | 暗黑/悬疑/夜景 |
| practical lights | 现场光（画面内灯具） | 台灯/霓虹/萤火/烛火 |
| motivated lighting | 有动机的光 | 有来源更真实 |
| volumetric light / god rays | 体积光/丁达尔 | 光束、氛围 |
| neon / candlelight / firelight | 霓虹/烛光/火光 | 色彩氛围 |
| **two color temperatures (warm vs cool)** | 冷暖双色温 | 电影感关键，必给 |

> 写法举例：`Rembrandt lighting, triangle of light on cheek, dramatic shadow, three-quarter angle light source`。

### 5. 风格 / 调色 / 镜头 / 胶片 Style block

| 类别 | 英文 token |
|---|---|
| 类型 genre | cinematic · photorealistic / hyper-detailed · **soft 3D storybook / Pixar-like**（少儿） · painterly / watercolor / hand-drawn（童话） · documentary / editorial / UGC · film noir / neo-noir · A24-style |
| 景深 depth | shallow depth of field / bokeh · deep focus |
| 镜头/光学 lens | wide-angle · telephoto / long lens · macro · 35mm/50mm spherical prime · anamorphic 2x · f/1.8 aperture · lens flare · rack focus |
| 质感/胶片 texture | film grain · halation · anamorphic lens flare · 16mm/35mm film stock · VHS / sepia 复古 |
| 调色 grade | teal-and-orange grade · monochrome / black-and-white · warm picture-book palette |
| 器材锚点 | shot on ARRI Alexa / Kodak Vision3 |

### 6. 画幅 / 时长 / 负面词 Format & Negative

- **画幅 aspect**：`16:9`（横/电影/横屏）· `9:16`（竖屏/手机）· `1:1`（方）· `2.39:1`（宽银幕，最电影感）
- **时长 duration**：Veo `4/6/8s`；Kling `5–10s`（部分版本 3–15s）；Sora `4/8/12/16/20s`。写法 `8-second clip` 或在参数面板设。
- **负面词 negative**（独立字段）：`blurry, distorted, extra fingers, warped face, deformed, text, watermark, flicker, jitter`
- **少儿加项**：`scary, violent, dark horror, sharp teeth, frightening`

### 示例 3 ·《萤火虫夜灯》（氛围、低调光教学）

全片 style block：`dreamy painterly 3D animation, magical fairytale night, low-key lighting, warm firefly glow vs cool blue moonlight (two color temperatures), volumetric haze, soft bokeh, gentle & safe` ｜ 5s ｜ 16:9

| 镜 | 景别 | 机位+运镜 | 主体 | 动作 | 场景 | 光线 |
|---|---|---|---|---|---|---|
| S1 | establishing wide | eye-level, slow push-in | 一点暖光在蕨叶间 | 微光在林间漂移 | 深蓝夜森林、薄雾 | 低调：冷蓝月光 + 单点暖光 |

```

S3: Medium shot, low angle, slow dolly-in, of a small lost baby bunny sitting alone in the dark grass, ears drooping, as the glowing firefly floats closer and lights up its face. ## 四、资源库（按用途分类 · 来源 + 链接 + 写法要点）
```

### A. 跨平台通用骨架 & 公式
- **Veo 3/3.1 提示词指南** — https://www.visla.us/blog/guides/how-to-prompt-veo-3-and-veo-3-1/
  要点：`[Cinematography]+[Subject]+[Action]+[Context]+[Style&Audio]`；支持 JSON 结构化（防 concept bleed）；时长 4/6/8s。
- **V.I.D.E.O. 五段式模板**（Medium @slakhyani20） — https://medium.com/@slakhyani20/ai-video-generation-prompt-template-system-you-can-use-696b4672dcc1
  要点：Visuals/Intent/Details/Emotion/Output 逐项填空，把「模糊创意」转成可逐条核对的硬指标，按需取用不必填满。
- **进阶电影级结构 & style block**（Envato Elements） — https://elements.envato.com/learn/advanced-ai-video-prompting
  要点：做固定 style block 贴每条 prompt；先文生图定「锚点」静帧再图生视频；precision over creativity（精确＞花哨）。

### B. 镜头语言（景别 / 机位 / 运镜）
- **Kling 镜头控制与运镜指南** — https://kling.ai/blog/ai-camera-control-movement-prompts-guide
  要点：景别/机位/运镜三者**分开写**互不污染；流程 `Camera→Subject&action→Environment→Lighting→Style→Constraints`，2–4 句最稳。
- **运镜大全**（Imagine.art） — https://www.imagine.art/blogs/ai-camera-movement-prompts
  要点：别写 `camera moves through scene`；给齐 type+speed+direction+起止；把 CAMERA 指令单独成段（全大写）并给「动机」motivated movement。
- **景别与机位角度**（Animation Guides） — https://www.animationguides.com/guide-camera-angles-shots-ai/
  要点：每种景别/角度承载情绪与权力（仰拍=强大、俯拍=弱小、Dutch=不安）；写法 `[景别/角度] of [主体]+[环境]+[光/风格]`。

### C. 官方权威指南
- **OpenAI Sora 2 Prompting Guide**（最权威） — https://developers.openai.com/cookbook/examples/sora/sora2_prompting_guide
  要点：五段式 Scene / Cinematography / Actions(编号 beat) / Dialogue / Ambient sound；Cinematography 单独成段；**时长画幅走 API 参数** `seconds(4/8/12/16/20)` + `size(720x1280 竖 / 1280x720 横；pro 加 1080p)`；适度留白鼓励创造力。
- **promptingguide.ai + Runway / Kling / LTX 官方博客 + LetsEnhance「12 种运镜」+ Travis Nicholson「100 cinematic AI prompts」** — 跨平台术语对齐与大量可套用范例（综合来源，按搜索摘要整合）。

### D. 光线 & 风格细化
- **AI 光线提示词术语**（Zsky） — https://zsky.ai/blog/ai-lighting-prompts
  要点：每条至少两种色温对比＝电影感来源；掌握 Rembrandt / chiaroscuro / split / rim / practical / volumetric / low-key·high-key / motivated。
- **Filmora / HumanAcademy 灯光术语** — 三点布光、轮廓光、体积光等基础（综合来源）。

### E. 中文平台（即梦等）
- **即梦 Jimeng 中文公式**（知乎） — https://zhuanlan.zhihu.com/p/1925872056056148876
  要点：基础＝主体+场景+运动；进阶＝主体+场景+运动+镜头语言+氛围词+风格化；**镜头语言在菜单栏勾选**，不必写进文本框；中英框架同构，跨平台**只换术语不换框架**。
